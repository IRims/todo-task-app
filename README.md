# My React Project

This is a simple React application.

## Installation

1. Clone this repository:
    
   git clone https://github.com/IRims/todo-task-app.git

2. Install dependencies:
    npm install
Usage 

3. Start server

    cd server
    npm start 
Open http://localhost:3000 in your browser.


4. Start client
    cd client
    npm run dev
